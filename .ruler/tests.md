# Test Instructions for Talkient Extension

## Constraints

- You MUST always run all tests after any change in the codebase.
- Use `pnpm test:all` as the default test command.

## Running Tests

1. Run all tests (unit tests twice and E2E tests four times): `pnpm test:all`
2. If `pnpm test:all` fails, fix the root cause and run it again until it passes.
